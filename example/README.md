## Getting Started

To run the example leptos app, trunk, as well as the node modules, must be installed.

### Prerequisites

- trunk

  ```sh
  cargo install trunk
  ```

- npm
  ```sh
  npm i
  ```

## Usage

Build and run the leptos app with:

```sh
trunk serve
```

Go to http://127.0.0.1:8080/
